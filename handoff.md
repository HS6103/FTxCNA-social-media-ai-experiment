## xlsx 欄位順序

`#` / `日期` / `主題` / `圖像觸及` / `連結觸及` / `觸及倍率` / `圖像互動` / `連結互動` / `連結點擊` / `分類` / `圖像ER%` / `連結ER%` / `ER差距(pp)` / `備註`

- ER% = 互動 / 觸及 × 100
- 持平定義：互動差距 ≤ 1
- 多變體取最佳圖像 vs 最佳連結（鼎泰豐、英代表、奈及利亞女孩、HIV 治療）
